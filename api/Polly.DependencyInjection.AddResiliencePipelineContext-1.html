Parameters

name string

The named options, if any.

Type Parameters

TOptions

The options type to listen to.

Remarks

You can decide based on the name to listen for changes in global options or named options. If name is null then the global options are listened to.

You can listen for changes from multiple options by calling this method with different TOptions types.
